# To-Do List using API (Python)

## Description
This project is a command line To-Do List application that uses a REST API
to perform CRUD operations like add, update, delete and view tasks.

## Features
- View Tasks
- Add Task
- Update Task
- Delete Task
- API Integration

## Technologies Used
- Python
- Requests Library
- REST API
- JSON

## How to Run
1. Install Python
2. Install requests library
3. Run main.py
